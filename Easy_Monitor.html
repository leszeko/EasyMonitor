Easy Monitor is set of SuperKaramba themes and simple bash scripts. Can display infos about your: distro, procesor, memory, disk usage, clock, network speed,.. and even do many: administrative, configuration, installation and backup tasks.

Features

  • Any of these themes you can edit and configure by lock its position and click on top frame of it
  • Most of the scripts you can edit and configure by click on corresponding configure button
  • Easy to build your own themes and script
